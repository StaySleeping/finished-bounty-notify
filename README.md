# Finished Bounty Notify

A RuneLite plugin that notifies you when you finish collecting all parts for a sailing bounty task.

Portions of this project are derived from [port-tasks](https://github.com/nucleon/port-tasks) by nucleon and Cooper Morris, used under the [BSD 2-Clause License](https://github.com/nucleon/port-tasks/blob/main/LICENSE). Affected source files include `BountyTaskCatalog.java`, `BountyTaskData.java`, `BountyTaskWidgetReader.java`, and `PortTaskTrigger.java`.

python3 scripts/generate-wiki-fallback.py
